Paragrafo<p>

Voltar

A tag <p> é utilizada para definir parágrafos no HTML. Ela é responsável por criar um bloco de texto, geralmente com uma margem vertical antes e depois para separá-lo de outros elementos na página, facilitando a leitura do conteúdo.

Links (<a>):

Voltar

A tag <a>, também conhecida como âncora, é usada para criar links em um documento HTML. Ela permite conectar uma parte do texto ou uma imagem a outra página da web, um arquivo, um email, ou mesmo um ponto específico em uma página através do atributo href.

Títulos (<hN>):

Voltar

As tags de títulos no HTML, que vão de <h1> a <h6>, são usadas para definir cabeçalhos de diferentes níveis de importância. <h1> representa o título mais importante e geralmente é usado para o título principal da página, enquanto <h6> representa o nível de cabeçalho menos importante.

Citação (<blockquote>):

Voltar

A tag <blockquote> é usada para indicar citações extensas de outras fontes. Textos dentro desta tag são geralmente apresentados com indentação, diferenciando-os do restante do conteúdo para indicar que o texto é uma citação de outra fonte.

Imagem (<img>):

Voltar

A tag <img> é utilizada para incorporar imagens em documentos HTML. Esta tag é vazia, ou seja, não possui tag de fechamento, e requer pelo menos o atributo src, que especifica o caminho para a imagem que será exibida na página web. Outros atributos como alt podem ser adicionados para fornecer texto alternativo caso a imagem não possa ser exibida.
